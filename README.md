# Проект: Место (Проектная работа №4)

Одностраничный сайт, адаптированный, с возможностью заполнения профиля и сохранения данных до перезагрузки страницы.

### Технологии

* JavaScript, CSS, HTML
* Файловая структура по БЭМ (метод Nested)
